# 🚀 Kubernetes Multi-Cluster GitOps Platform

A production-style GitOps platform built using Kubernetes and Argo CD to manage applications across multiple clusters (dev, staging, production).

## 🔧 Tech Stack

* Kubernetes (EKS / AKS)
* Argo CD (GitOps)
* Terraform (Infrastructure as Code)
* Helm (Templating)
* GitHub Actions (CI/CD)

## 🌍 Key Features

* Multi-cluster deployment strategy
* Automated GitOps-based delivery
* Environment promotion (dev → staging → prod)
* Infrastructure provisioning using Terraform
* Scalable and resilient architecture

## ⚡ How It Works

1. Developer pushes code to GitHub
2. CI pipeline builds and updates manifests
3. Argo CD detects changes
4. Automatically syncs to Kubernetes clusters
